# 🚀 SkillMatch | The Bias-Free Equity Marketplace

**Built for the DEV 2026 Gender Equity Hackathon** 🏆

## 🥋 The Mission
As an 11-year-old developer and 1st Dan Taekwondo Black Belt, I believe that **strength and talent** are what matter most. In the tech world, "unconscious bias" can sometimes stop girls and underrepresented voices from being seen. 

**SkillMatch** is a full-stack platform designed to close the gender gap by using **Blind-Hiring Logic**. Recruiters search by **Skills First**, ensuring that every developer is judged by their code, not their gender or name.

---

## ✨ Key Features
* **Skill-First Search:** A professional search engine that prioritizes tech stacks over personal identifiers.
* **Dynamic Talent Board:** Users can "Login" and post their own skills to a global database in real-time.
* **One-Click Connection:** Integrated with the **WhatsApp Public API** to bridge the gap between recruiters and talent instantly.
* **Modern UI/UX:** A high-end "Glassmorphism" dashboard built with Tailwind CSS.

---

## 🛠️ Tech Stack
This project is a complete **Full-Stack Application** built from scratch:

| Layer | Technology |
| :--- | :--- |
| **Backend** | Python 3.14 + FastAPI Library |
| **Server** | Uvicorn (ASGI) |
| **Frontend** | HTML5 + JavaScript (ES6+) |
| **Styling** | Tailwind CSS (Glassmorphism Design) |
| **API** | WhatsApp Deep Linking API |
